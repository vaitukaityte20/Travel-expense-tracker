# Travel Budget Tracker App

## Features
- Add, edit, and delete expenses directly in the table  
- Manage and remove categories dynamically  
- Live-updating summary cards with progress bars  
- Visual indicators for over-budget categories  
- Confirmation modals for safe deletions  
- Instant feedback messages for all user actions  
- Data saved automatically in local storage  
- Fully responsive design for all devices  

## Technologies
